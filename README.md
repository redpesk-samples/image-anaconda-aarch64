# Build your redpesk OS image (aarch64) using anaconda

redpesk factory allows to create redpesk OS image with your custom applications and settings based on anaconda tool.

Anaconda is a Fedora installer tool that generates disk images using kickstart syntax.

For more details, please refer to

- online redpesk documentation on how to create an image is available at [https://docs.redpesk.bzh](https://docs.redpesk.bzh/docs/en/master/redpesk-factory/images-management/01-create-an-image.html)

- Kickstart syntax Reference available at [https://docs.fedoraproject.org](https://docs.fedoraproject.org/en-US/fedora/f36/install-guide/appendixes/Kickstart_Syntax_Reference/)
